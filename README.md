# COMPLETE FORMULA SHEET

## Chapter 1: Similarity

### Basic Proportionality Theorem (BPT)
```
If DE || BC in ΔABC, then:
AD/DB = AE/EC
```

### Ratio of Areas of Similar Triangles
```
If ΔABC ~ ΔPQR, then:
A(ΔABC)/A(ΔPQR) = (AB/PQ)² = (BC/QR)² = (AC/PR)²
```

### Angle Bisector Theorem
```
In ΔABC, if AD bisects ∠A, then:
BD/DC = AB/AC
```

---

## Chapter 2: Pythagoras Theorem

### Pythagoras Theorem
```
In right-angled ΔABC (∠B = 90°):
AC² = AB² + BC²
```

### Converse
```
If AC² = AB² + BC², then ∠B = 90°
```

### Common Pythagorean Triplets

(3, 4, 5), (5, 12, 13), (8, 15, 17)
(7, 24, 25), (9, 40, 41), (20, 21, 29)
```

## Chapter 3: Circle

### Inscribed Angle Theorem
```
Inscribed angle = 1/2 × Measure of intercepted arc
```

### Central Angle
```
Central angle = Measure of intercepted arc
```

### Tangent-Secant Theorem
```
PS² = PQ × PR
(where PS is tangent, PQR is secant)
```

### Intersecting Chords
```
AE × EB = CE × ED
```

### Cyclic Quadrilateral
```
Opposite angles are supplementary
∠A + ∠C = 180°, ∠B + ∠D = 180°
```

## Chapter 4: Co-ordinate Geometry

### Distance Formula
```
d = √[(x₂-x₁)² + (y₂-y₁)²]
```

### Midpoint Formula
```
M = [(x₁+x₂)/2, (y₁+y₂)/2]
```

### Slope Formula
```
m = (y₂-y₁) / (x₂-x₁)
```

### Section Formula
```
P = [(mx₂+nx₁)/(m+n), (my₂+ny₁)/(m+n)]
```

### Collinearity Check
```
Slope of AB = Slope of BC
```

## Chapter 5: Trigonometry

### Basic Identities
```
sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = cosec²θ
```

### Reciprocal Relations
```
cosec θ = 1/sin θ
sec θ = 1/cos θ
cot θ = 1/tan θ
```

### Standard Values
```
sin 30° = 1/2,      cos 30° = √3/2,     tan 30° = 1/√3
sin 45° = 1/√2,     cos 45° = 1/√2,     tan 45° = 1
sin 60° = √3/2,     cos 60° = 1/2,      tan 60° = √3
```

### Complementary Angles
```
sin(90°-θ) = cos θ
cos(90°-θ) = sin θ
tan(90°-θ) = cot θ
```

# CHAPTER-WISE DETAILED ANALYSIS

## Chapter 1: Similarity

### Weightage Analysis
- **Average Marks**: 10.0 per year
- **Weightage**: 25.0%
- **Priority**: HIGH

### Important Topics

#### 1. Ratio of Areas (Every Year)
**Formula**: Ratio of areas = (Ratio of sides)²

**Example**:
```
If AB:PQ = 2:3, then:
A(ΔABC):A(ΔPQR) = 4:9
```

#### 2. Basic Proportionality Theorem
**Statement**: If a line is parallel to a side of a triangle, it divides the other two sides proportionally.

**Example**:
```
In ΔABC, DE || BC
AD = 6, DB = 9, EC = 7.5
Find AE.

Solution:
AD/DB = AE/EC
6/9 = AE/7.5
AE = (6 × 7.5)/9 = 5
```

#### 3. Angle Bisector Theorem
**Statement**: The angle bisector divides the opposite side in the ratio of adjacent sides.

### Common Mistakes
1. Using ratio of sides instead of ratio of areas squared
2. Not identifying corresponding sides correctly
3. Calculation errors in proportions

## Chapter 2: Pythagoras Theorem

### Weightage Analysis
- **Average Marks**: 9.4 per year
- **Weightage**: 23.6%
- **Priority**: HIGH

### Important Topics

#### 1. Pythagorean Triplets (Most Repeated)
**Common Triplets**:
```
(3, 4, 5)    → 3² + 4² = 5²
(5, 12, 13)  → 5² + 12² = 13²
(8, 15, 17)  → 8² + 15² = 17²
(7, 24, 25)  → 7² + 24² = 25²
(9, 40, 41)  → 9² + 40² = 41²
```

#### 2. Finding Hypotenuse/Sides
**Example**:
```
In right-angled ΔABC, ∠B = 90°
AB = 6, BC = 8. Find AC.

AC² = AB² + BC² = 36 + 64 = 100
AC = 10
```

#### 3. Median on Hypotenuse
**Property**: Median on hypotenuse = 1/2 × Hypotenuse

### Common Mistakes

## Chapter 3: Circle

### Weightage Analysis
- **Average Marks**: 10.9 per year
- **Weightage**: 27.1% (HIGHEST)
- **Priority**: HIGHEST

### Important Topics

#### 1. Inscribed Angle Theorem (Most Repeated)
**Theorem**: Inscribed angle = 1/2 × Measure of intercepted arc

**Example**:
```
If ∠ACB = 65° (inscribed in arc ACB)
Then m(arc ACB) = 2 × 65° = 130°
```

#### 2. Tangent-Secant Theorem
**Theorem**: PS² = PQ × PR

**Example**:
```
PQ = 3.6, QR = 6.4
PS² = PQ × PR = 3.6 × (3.6 + 6.4)
PS² = 3.6 × 10 = 36
PS = 6
```

#### 3. Cyclic Quadrilateral
**Property**: Opposite angles are supplementary

**Example**:
```
In cyclic quadrilateral ABCD
∠B = 110°
∠D = 180° - 110° = 70°
```

### Common Mistakes
1. Confusing inscribed angle with central angle
2. Forgetting tangent-secant formula
3. Wrong arc measurement

## Chapter 4: Geometric Constructions

### Weightage Analysis
- **Average Marks**: 8.1 per year
- **Weightage**: 20.4%
- **Priority**: MODERATE (but 100% appearance)

### Important Constructions

#### 1. Tangent from External Point (Every Year)
**Steps**:
1. Draw circle with centre O and radius r
2. Mark external point P at given distance
3. Join OP
4. Find midpoint M of OP
5. Draw circle with centre M and radius OM
6. Mark intersection points A and B
7. Join PA and PB - these are tangents

#### 2. Tangent at Point on Circle
**Steps**:
1. Draw circle with centre O
2. Mark point P on circle
3. Join OP
4. Draw perpendicular to OP at P
5. This is the tangent

#### 3. Similar Triangle Construction
**Steps**:
1. Draw given triangle
2. Mark point for new triangle
3. Draw ray at given angle
4. Cut arc at scaled distance
5. Complete triangle

### Common Mistakes
1. Not leaving construction marks
2. Using pen instead of pencil
3. Not labeling points
4. Erasing construction arcs

## Chapter 5: Co-ordinate Geometry

### Weightage Analysis
- **Average Marks**: 7.3 per year
- **Weightage**: 18.2% (LOWEST)
- **Priority**: MODERATE

### Important Topics

#### 1. Slope (Every Year)
**Formula**: m = (y₂-y₁) / (x₂-x₁)

**Example**:
```
A(2, 3), B(4, 7)
Slope = (7-3)/(4-2) = 4/2 = 2
```

#### 2. Collinearity Check
**Method**: Slope of AB = Slope of BC

**Example**:
```
A(1, -3), B(2, -5), C(-4, 7)
Slope AB = (-5+3)/(2-1) = -2
Slope BC = (7+5)/(-4-2) = -2
Since equal, points are collinear
```

#### 3. Distance Formula
**Formula**: d = √[(x₂-x₁)² + (y₂-y₁)²]

### Common Mistakes
1. Sign errors in coordinates
2. Wrong order of points
3. Calculation errors in squares

#### 2. Standard Values (Must Memorize)
```

**Example**:
```
Observer at 10m from tree, angle of elevation = 60°

tan 60° = h/10
√3 = h/10

```
